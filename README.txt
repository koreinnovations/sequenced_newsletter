
-- SUMMARY --

Sequenced newsletter takes a category of Simplenews newsletters and
sends it's emails in a sequenced intervals. Administrator can set interval 
between each message. New subscriber will get email every time when this
interval finishes. It will stop receiving emails when recieved all of them.

For a full description of the module, visit the project page:
  http://drupal.org/project/sequenced_newsletter


-- REQUIREMENTS --

Module requires Simplenews and Date modules to be enabled. 


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Customize the module settings in Administer >> Site configuration >>
  Sequenced newsletter.
